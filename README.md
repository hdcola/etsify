# Etsify Web

## Development

### Setup

```bash
pnpm install
```

### Run the development server

```bash
pnpm dev
```

### Run tests

Install [Vitest](https://marketplace.visualstudio.com/items?itemName=vitest.explorer) in your VSCode and run the tests from the test explorer.

Run the tests in CLI:

```bash
pnpm test
```
